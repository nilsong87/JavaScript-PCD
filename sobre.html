Sobre o Projeto

Nossa Missão

O JavaScript Acessível nasceu da necessidade de tornar o aprendizado de programação verdadeiramente inclusivo para pessoas com deficiência visual.

Nosso objetivo é eliminar as barreiras que impedem o acesso igualitário ao conhecimento de desenvolvimento web.

Nossa História

Fundado em 2023 por Nilson Gomes, o projeto começou como uma iniciativa pessoal e cresceu para se tornar uma referência em educação tecnológica inclusiva.

Hoje contamos com uma equipe multidisciplinar dedicada à acessibilidade digital.

Princípios de Acessibilidade

Perceptível

Todo conteúdo é apresentado de formas que possam ser percebidas por todos os usuários.

Operável

Interface navegável por teclado, com tempo suficiente para leitura e uso.

Compreensível

Conteúdo legível e previsível, com assistência para correção de erros.
